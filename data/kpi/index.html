📊 KPI Master Dataset

Master Dataset: This directory contains the consolidated KPI dataset with all education metrics in standardized long format. Each row represents one metric for one student group at one school/district for one year.
  • 📄 kpi_master.csv 175.1 MB

📋 Dataset Schema

The KPI master file contains exactly 10 columns:

  • district - Kentucky district name
  • school_id - Unique school identifier (NCES/state ID)
  • school_name - Human-readable school name
  • year - 4-digit academic year
  • student_group - Standardized demographic group
  • metric - Metric name following {indicator}_{type}_{period} convention
  • value - Numeric value (NaN when suppressed)
  • suppressed - "Y" or "N" suppression flag
  • source_file - Original filename for audit trail
  • last_updated - ISO timestamp

🎯 Included Metrics

  • Chronic Absenteeism Rates - Percentage of students missing 10% or more school days
  • English Learner Progress Rates - Progress toward English proficiency
  • Graduation Rates - 4-year high school graduation rates
  • Kindergarten Readiness Rates - School readiness composite scores
  • Out of School Suspension Rates - Student discipline rates
  • Postsecondary Enrollment Rates - College enrollment after graduation
  • Postsecondary Readiness Rates - College and career readiness

Note: All suppressed records are included with NaN values to maintain transparency. Data covers multiple years with standardized demographic mappings.
