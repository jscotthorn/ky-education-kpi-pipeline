📁 Raw Data Files

Original Source Data: These directories contain files from the Kentucky Department of Education School Report Card Datasets. Excel files have been converted to CSV format. Files maintain original column names and formatting.
  • 📂 chronic_absenteeism/ Student attendance and chronic absenteeism source files
  • 📂 english_learner_progress/ English language proficiency progress source files
  • 📂 graduation_rates/ High school graduation rate source files
  • 📂 kindergarten_readiness/ Kindergarten screening and readiness source files
  • 📂 out_of_school_suspension/ Student discipline and suspension source files
  • 📂 postsecondary_enrollment/ College enrollment tracking source files
  • 📂 postsecondary_readiness/ College and career readiness source files

📋 Data Sources

  • KYRC24 Files - Kentucky Report Card 2024 official data releases
  • Historical Files - Previous years' data for trend analysis (2021-2023)
  • Original Formatting - Unmodified column names and data structures

Note: Raw files are preserved exactly as received from data sources. See the processed/ directory for cleaned, standardized versions suitable for analysis.
