📁 ky-education-kpi-pipeline / data
Education Data Portal: This directory contains education performance indicator data for Kentucky schools and districts. Data is organized by metric type with both raw source files and processed KPI outputs following standardized formatting for transparency and analysis.

📂 Data Directories

  • 📊 kpi/ Master KPI dataset with all metrics in standardized format
  • ⚙️ processed/ Cleaned and standardized metric files ready for analysis
  • 📁 raw/ Original data files from Kentucky Department of Education

📋 Available Metrics

  • Chronic Absenteeism - Student attendance patterns
  • English Learner Progress - English language proficiency development
  • Graduation Rates - 4-year high school graduation outcomes
  • Kindergarten Readiness - School readiness screening results
  • Out of School Suspension - Student discipline data
  • Postsecondary Enrollment - College enrollment after graduation
  • Postsecondary Readiness - College and career readiness indicators

Note: All data files follow standardized KPI format with demographic breakdowns and suppression handling for student privacy.
